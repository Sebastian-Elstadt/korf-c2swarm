# 2026-03-02

## Nodus

### General notes

I've opted for utilising Base85 throughout the project instead of Base64. Base85 produces a much smaller result, and I intend to implement my own simple binary protocol for communication between the node and the C2 server. Or perhaps some existing implementation of protocol buffers... Will get to this later.

Base85 does produce challenges with it's larger character set, which isn't a problem if I opt for binary communication. Base85 might stand out as unusual, but only to someone who is looking for that. Will keep this all in mind.

### anti_analysis.rs

Added a new function to test internet access. The node program cannot operate offline.

### identity.rs

New file which allows the node to scrape identifying info of the host device. It also hashes this info to create its own unique identifier. However, I am not too confident in the uniqueness, so I will be adding some extra randomness either inside the hash, or by appending a UUID or similar.
